# Отель «Морская Жемчужина» — сайт-визитка

Сайт-визитка для небольшого отеля: главная страница, каталог номеров и форма
бронирования без предоплаты (имя + телефон).

## Стек

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- lucide-react (иконки)

## Запуск

```bash
npm install
npm run dev
```

Сайт будет доступен на [http://localhost:3000](http://localhost:3000).

## Структура

- `app/page.tsx` — главная (визитка): hero, об отеле, услуги, популярные номера.
- `app/rooms/page.tsx` — каталог всех номеров с формой бронирования у каждого.
- `app/api/booking/route.ts` — приём заявок (сейчас просто логирует в консоль).
- `lib/rooms.ts` — список номеров (легко редактировать).
- `components/` — `Header`, `Footer`, `BookingForm`.

## Что доделать в проде

- Подключить отправку заявок: email / Telegram-бот / CRM (Bitrix24, amoCRM).
- Заменить демо-фотографии номеров на реальные.
- Обновить контакты и адрес в `components/Footer.tsx`.
- Добавить страницу политики конфиденциальности.
